Timeout problem with cuda on different OS:


1. Windows
PROBLEM: Timeout Detection and Recovery of GPUs (work only 5 sec)
ANSWER: disable TDR (http://msdn.microsoft.com/en-us/windows/hardware/gg487368.aspx)

2. Ubuntu 
PROBLEM: kernel function return error: "the launch timed out and was terminated"
ANSWER: ?don't work? set X to runlevel 3 (http://askubuntu.com/questions/228402/boot-to-runlevel-3)
